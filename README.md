Project Title: Custom Coded Commercial Cleaning Website
Role: Front-End Developer & Web Designer

About This Project:
I designed and developed this custom, high-performance landing page for a commercial cleaning business in Sydney. The goal was to create a premium, trustworthy digital presence that appeals to B2B clients (facility managers, clinics, and corporate offices).

Technical Highlights:

Custom Vanilla Code: Built from the ground up using semantic HTML5, CSS3 Grid/Flexbox, and vanilla JavaScript without relying on heavy page builders.
Flawless Performance: Achieved perfect Google Lighthouse scores (99 Performance, 94 Accessibility, 100 Best Practices, 100 SEO), ensuring lightning-fast load times.
Responsive Design: Fully mobile-optimized with a custom hamburger menu and fluid typography that adapts to all screen sizes.
Modern UI/UX: Features interactive CSS animations, scroll-triggered fade-ins (Intersection Observer), and a clean, high-conversion layout designed to drive quote requests.
